# 📝 To Do Task App

A simple and responsive To Do Task web application built using HTML, CSS, and JavaScript. This application helps users add, manage, and organize their daily tasks with an easy-to-use interface.

---

## 📸 Preview

![To Do Task App](todo.png)

---

## ✨ Features

- ➕ Add new tasks
- 📋 Display task list
- ✔️ Mark tasks as completed
- 🗑️ Delete tasks
- 📱 Responsive user interface
- 🎨 Simple and attractive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/todo-task-app.git
```

### Open the Project

```bash
cd todo-task-app
```

### Run the Application

Open the **index.html** file in your web browser.

No installation is required.

---

## 📖 How to Use

1. Enter a task in the input box.
2. Click the **ADD** button.
3. Your task will appear in the task list.
4. Mark tasks as completed when finished.
5. Delete tasks whenever needed.
